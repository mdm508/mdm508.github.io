What is a information

Anything you can sense is information.

  • music
  • colors
  • the name of school

Basically information is stuff in the real world.

You cant literally put this kind of information into computer.

01001010100010 name of an mp3 number number of kids in your school ip adress

We can represent things in the real world in a form the computer can understand.

Shoe

Brand, Price, Qty, Color, ID, Model, Location

using the innformation we would like to answer some questions

  • check if a shoe is in stock
  • check what is the most expensive shoe
  • which store is in stock

Atomic

This is when information can be captured by a single atomic value

  • atomic means the most basic type of data
  • Number String Image Boolean

Sometimes we have information that we can simply represent as one of these atomic types

Brand is Number Price is Number

“nike” ——> 010101

Interval

Your information is easy to represent by a number and there are limits on what that number can be.

ValidHeight is a Number > 5
