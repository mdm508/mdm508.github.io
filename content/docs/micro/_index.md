Without microprocessors, our computers would not be able to do very interesting stuff.
This page is dedicated to helping you learn the basics of processors and computer
hardware.

## Learn with Manga
These pages will help you to review and remember the essential ideas found in
"The Manga Guide to Microprocessors". This book teaches you about how computers work
using storytelling and pictures. It is a lot of fun. My goal with this page is to summarize
some of the essential points in the book and help you to remember them.
